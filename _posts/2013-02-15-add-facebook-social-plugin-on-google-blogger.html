1. 코드 만들기

아래 코드는 가장 기본적인 좋아요!+보내기가 있는 페북 공유 플러그인의 코드입니다.

<div>
<b:if cond='data:post.isFirstPost'> 
<script>(function(d){
  var js, id = 'facebook-jssdk'; if (d.getElementById(id)) {return;}
  js = d.createElement('script'); js.id = id; js.async = true;
  js.src = "//connect.facebook.net/en_US/all.js#xfbml=1";
  d.getElementsByTagName('head')[0].appendChild(js);
}(document));</script>
</b:if> 
<fb:like expr:href="data:post.canonicalUrl" layout='standard' send='true' show_faces='false' font="arial" action="like" colorscheme="light"></fb:like>
</div>

여기에서 여러가지 옵션을 더하고 뺄수 있습니다. ( 꼭 수정할 필요는 없어요. )

layout='standard' 이것은 좋아요! 숫자를 나타내고, layout='button_count'로 설정할경우 박스형태로 나타냅니다.

send='true' 이것은 보내기 버튼의 유무를 결정, 'false'로 하면 보내가가 없어집니다.

show_faces='false' 이것은 같이 좋아요!를 한 친구들의 얼굴이 나오는 것인데, 'true'일경우 얼굴들이 나옵니다.

2.적용

설정 - 템플릿 - HTML편집에 들어간후 상단에 '가젯코드 펼쳐서 보여주기'를 체크합니다.
그리고 <data:post.body/>를 찾아서, 가장 처음 나온 <data:post.body/>아래에 위의 코드를 붙입니다.

이제 HTML 네임스페이스라는 형식을 추가할 차례입니다.

HTML가장 윗부분에 <html 이라는 것이 있을겁니다.
이것을<html xmlns:fb="https://www.facebook.com/2008/fbml" 로 변경하시면 됩니다. 간단히 <html 포함해서 덮어씌우면 됩니다.

그리고 지금까지 수정한것을 저장하면 끝입니다!

블로그로 돌아가 확인해보면 좋아요! 버튼이 붙어있을겁니다.
